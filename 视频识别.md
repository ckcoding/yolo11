**请求方式**

> POST

**Content-Type**

```javascript
{
	"stream_url": "http://192.168.3.231:8080/live/1.flv",
```

| 参数名 | 示例值 | 参数类型 | 是否必填 | 参数描述 |
| --- | --- | ---- | ---- | ---- |
| m3u8_url | rtmp://192.168.3.231/live/abcd | string | 否 | rtmp的url地址 |
| names_dict | pedestrian,people,bicycle,car,van,truck,tricycle,awning-tricycle,bus,motor | string | 否 | 分别指：行人, 人, 自行车, 汽车, 面包车, 卡车, 三轮车, 带篷三轮车, 公交车, 摩托车 |
| alias | zhangsan | string | 否 | 别名，区别谁是谁，选传 |
| confidence_display | true | boolean | 否 | 是否显示置信度，默认不显示 |
| conf_thres | 0.25 | float | 否 | 置信度阈值，默认0.25 |

> 继承父级

| 参数名 | 示例值 | 参数类型 | 是否必填 | 参数描述 |
